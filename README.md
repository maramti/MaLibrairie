# MyBookDB – REST API en Node.js & MongoDB

Une API REST simple permettant de gérer une collection de livres  
(Ajout, suppression, modification, récupération).

Ce projet a été développé en utilisant **Node.js**, **Express**, et **MongoDB Atlas**.

---

## Fonctionnalités

- Ajouter un livre  
- Supprimer un livre  
- Modifier le statut d’un livre  
- Obtenir tous les livres  
- Gestion d’erreurs (duplicate key, document introuvable, etc.)

---

## Technologies utilisées

- **Node.js**
- **Express.js**
- **MongoDB Atlas**
- **Nodemon** (dev)
- **Body-parser** ou `express.json()`
